## SeceumFL


SeceumFL是一个基于Web的联邦学习全生命周期的管理平台，在[FATE](https://github.com/FederatedAI)的基础上，对它进行了改进，并提供了用户友好的界面，旨在降低用户使用联邦学习的门槛，帮助用户轻松开展联邦建模任务。用户可以通过可视化的方式进行数据管理、建模、模型管理等工作。
