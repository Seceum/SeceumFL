黤黕凝黛色，峥嵘当曙空
黯与山僧别，低头礼白云
鼓击春雷，直破烟波远远回
鼹鼠饮河，不过满腹
齐安郡中偶题·其一
齐鲁青未了
龙吟虎啸，凤翥鸾翔，大丈夫之气象
龙头舴艋吴儿竞，笋柱秋千游女并
龙蛇四海归无所，寒食年年怆客心
